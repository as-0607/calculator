# 🧮 Calculator

A modern and responsive calculator built with **HTML**, **CSS**, and **JavaScript**. The project performs basic arithmetic operations with a clean dark-themed interface inspired by modern calculator designs.

## 📸 Preview

![Calculator Preview]([./preview.png](https://as-0607.github.io/calculator/))

---

## ✨ Features

- Clean and responsive UI
- Addition, subtraction, multiplication, and division
- Decimal number support
- Clear (AC) button
- Displays both the entered equation and the result
- Error handling for invalid expressions
- CSS Grid layout for the calculator buttons
- Smooth hover and click animations

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/calculator.git
```

2. Navigate to the project folder

```bash
cd calculator
```

3. Open `index.html` in your browser.

No installation or dependencies are required.

---

## 💻 How It Works

- Clicking a button appends its value to the equation display.
- Pressing **=** evaluates the mathematical expression using JavaScript.
- The result is displayed below the equation.
- Invalid expressions display an error message.
